## O que e o PRScripts?

O PRScripts (tambem chamado de **Parana Scripts**) e o **equivalente paranaense** do que scripts populares fazem em outros estados: automatiza as atividades das plataformas educacionais oficiais do Parana com IA, direto pelo navegador — **sem instalar extensao**.

Diferente de scripts genericos feitos para outros estados, o PRScripts foi treinado especificamente para o **Desafio Parana no Wayground**, o **Leia Parana**, a **Redacao Parana**, o **Alura Start** e o **Ingles Parana (EF Efekta)**. Reconhece o login `@escola.pr.gov.br` e valida licenca no proprio email.

## Como usar

1. Acesse [prscripts.app/planos](https://prscripts.app/planos) e escolha um plano (semanal, mensal, trimestral ou anual).
2. Ative sua licenca com o email `@escola.pr.gov.br`.
3. Arraste o bookmark de [prscripts.app/instalar](https://prscripts.app/instalar) para a barra de favoritos.
4. Entre na plataforma (Wayground, Leia, Alura, Khan, EF) e clique no bookmark.
5. Um menu flutuante aparece — clique em "Resolver" e a IA responde.

Para **Redacao Parana** e **Alura**, existe um **painel proprio** em subdominio dedicado — voce nem precisa do bookmark.
